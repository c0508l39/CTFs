Therefore, to extract the flag, we use the following script:

```python
from pwn import *

bin_str = ""
for i in range(5, 0, -1):
    with open("Item0{}_cp.bmp".format(i), "rb") as b:
        b.seek(2019)
        
        for j in range(50):
            if ((j % 5) == 0):
                for k in range(8):
                    bin_str += str(ord(b.read(1)) & 1)
            else:
                b.read(1)

char_str = unbits(bin_str, endian = 'little')
print char_str
```
